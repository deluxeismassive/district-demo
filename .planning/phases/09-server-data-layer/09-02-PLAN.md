### server/api/vendors.get.ts — COPY VERBATIM (research §1, §"Code Examples")

```ts
import vendors from '~~/server/data/vendors'

export default defineEventHandler(() => {
  return vendors
})
```

Critical notes (research §1, §10):
- `defineEventHandler` is auto-imported by Nuxt 4/Nitro — DO NOT add `import { defineEventHandler } from 'h3'`.
- Return value MUST be a plain object/array. DO NOT use `new Response(JSON.stringify(...))` — breaks useFetch type inference (research Pitfall #2).
- The `~~` alias resolves to the project root in Nuxt 4.
- The `.get.ts` suffix scopes this handler to GET method only (REST-correct, matches STATE.md locked decision).
- Nitro auto-serializes to JSON with `Content-Type: application/json` — no manual header setting needed.
